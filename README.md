## ***Бургерная на краю Вселенной - собери и закажи фантастический и неповторимый бургер***

*многопользовательское и многостраничное приложение интрнет-магазина, с возможностью собрать бургер из предлагаемых интгредиентов и сделать заказ. У пользователей есть возможность зарегистрироваться, авторизоваться, восстановить пароль, изменить свои данные в профиле, посмотреть историю своих заказов, а так же видеть все заказы, сделанные другими пользователями, посмотреть детали заказов и ингредиента. Приложение сверстанно в Desctop версии (1280px - 1080px)*
